1 功能说明

APP BLE蓝牙通讯部分功能, 封装API接口供业务层调用;

1.1 主要功能

  • 多个蓝牙智能设备管理;
  • 蓝牙数据命令发送与接收;
  • 蓝牙断开重连, 异常等处理;
  • 蓝牙设备搜索;

2 API接口

SDK 提供的 API 接口，供APP业务层使用;

2.1 REBtDeviceManager类

管理设备连接, 数量, 解绑删除; 单例类 REBtDeviceManager.I() 获取对象.

API - registerContext()

注册上下文与数据接收接口; 需最早调用;

接口定义
public void registerContext(Context appContext, REBtDataReceiveCallBack btDataCallBack)
参数说明
  • appContext Application上下文;
  • btDataCallBack REBtDataReceiveCallBack接口类, 连接状态, 电量与测试数据同步回调;
public interface REBtDataReceiveCallBack {



    //蓝牙状态改变回调

    public void btConnectStateChange(int state);



    //60s请求1次电量信息回调

    public void btDeviceBatteryChange(int battery);



    //各通道数据采集数据回调

    public void btDeviceTestDataReceive(ArrayList<Object> channelData, REBtDevice btDevice);

    //Object包含通信索引,本次采集时间, 采集值; 使用字典?

}
返回值

无

API - getDeviceBattery()

获取设备当前电量,同步方法; 每60s更新一次的电量值也会通过CallBack回调出去;

接口定义
public int getDeviceBattery()
参数说明

无

返回值

int 第一个字节为电量值0~100, 第二个字节为充电状态(0: 未充电 1: 充电完成 2: 正在充电);

int tBattery = curDeivce.getDeviceBattery();

int tBatteryValue = (tBattery & 0xff);

int tBatteryState = ((tBattery >> 8) & 0xff);

API - getDeviceWorkState()

读取设备（工作）状态请求(key 0x03);

接口定义
public int getDeviceWorkState()
参数说明

无

返回值

1: 设备正在工作测试；
2: 设备正在暂停测试；
0: 设备测试结束 待机；

API - startTest()

开始数据采集测试, 同步方式; 此中会执行6条命令,都成功后才会返回成功;

接口定义
public int startTest(ArrayList<REChannelItem> channelItems, String test_id, String app_user_id, String device_user_id)
参数说明
  • channelItems 8个通道的配置Item(包括通道使能, 通道采样间隔, 通道保存与发送设置), 通道序号对应数组序号
  • test_id 测试ID
  • app_user_id 登录用户ID
  • device_user_id 设备使用ID(设备连接时弹框会设置)
返回值
private static final int RETB_SEND_OK = 0;

private static final int RETB_SEND_Failed = 1;

private static final int RETB_SEND_Timeout = 1;

API - startOTAUpdate()

开始设备OTA升级, 同步返回是否开始成功;

接口定义
public int startOTAUpdate(byte[] otaFileData, REOTAProgressCallBack otaCallBack)
参数说明
  • otaFileData 要更新的OTA文件byte数组
  • otaCallBack REOTAProgressCallBack, OTA文件更新过程中的 回调反馈
public interface REOTAProgressCallBack {

    //升级完成

    public void otaProgressFinished();

    //升级失败

    public void otaProgressFailed();

    //升级过程中的进度

    public void otaProgressLoading(int ratio);

}
返回值

见RETB_SEND_xxx 常量定义
